Skills live as flat folders directly under `skills/` (no bucket categorization for now).

Every skill must have:

- An entry in the top-level `README.md` reference list, with the skill name linked to its `SKILL.md`.
- An entry in `.claude-plugin/plugin.json`.

Each skill folder contains a `SKILL.md` with YAML frontmatter (`name`, `description`). See `skills/writing-skills/SKILL.md` for the conventions on structure, descriptions, and progressive disclosure.
